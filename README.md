# أبو الزهراء للاتصالات - تطبيق VoIP

تطبيق اتصالات صوتي حديث باستخدام WebRTC و Twilio، مصمم خصيصاً للسوق السعودي والشرق الأوسط.

## 🌟 المميزات

- 📞 اتصالات صوتية عالية الجودة عبر WebRTC
- 🌍 دعم دولي مع أسعار تنافسية
- 👥 دفتر عناوين ذكي مع المفضلة
- 💰 رصيد حساب وعمليات دفع آمنة
- 🔐 مصادقة آمنة مع JWT
- 📱 واجهة مستخدم حديثة وسهلة الاستخدام
- 🌐 دعم متعدد اللغات (العربية والإنجليزية)

## 🏗️ البنية التقنية

### Frontend
- **Framework**: React Native with Expo
- **State Management**: Zustand
- **Navigation**: React Navigation
- **UI Components**: Custom components with TailwindCSS
- **Real-time Communication**: WebRTC
- **Storage**: AsyncStorage & SecureStore

### Backend
- **Framework**: FastAPI
- **Database**: PostgreSQL with SQLAlchemy
- **Authentication**: JWT
- **Payment**: Stripe
- **Communication**: Twilio
- **Documentation**: OpenAPI/Swagger

## 📋 المتطلبات

- Node.js 18+
- Python 3.9+
- PostgreSQL 13+
- Expo CLI
- Docker (اختياري)
